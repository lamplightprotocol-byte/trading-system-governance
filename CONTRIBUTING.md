# Contributing

Thanks for your interest.

This repository is primarily an informational landing page for the LEP v1 governance framework.
It is not an open-source codebase and does not contain executable trading code.

## What we accept

Small fixes are welcome via Pull Requests:
- typos
- broken links
- markdown formatting fixes

## What we do not accept

- feature requests
- trading strategy discussion
- requests for support / consulting
- code contributions for trading systems

If you want the full LEP v1 pack, see:
https://lamplight88.gumroad.com/l/lep-v1
